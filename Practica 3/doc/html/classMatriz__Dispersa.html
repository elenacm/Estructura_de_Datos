template<class T>
class Matriz_Dispersa< T >

TDA Matriz Dispersa.

Un objeto de tipo matriz_dispersa es un array 1-d de tripletas formadas por: una fila, columna y el valor asociado de tipo carácter. Solamente se almacenan los valores con un valor diferente a un valor por defecto (0)

{(i0, j0, mi, j),(i1, j1, mi, j),...,(in−1, jn−1, min−1,jn−1),(∗, d)}
