template<class T>
class Vector_Dinamico< T >

T.D.A. Vector_Dinamico.

Una instancia c del tipo de datos abstracto Vector Dinámico es una suscesión de elementos almacenados y accesibles en memoria. Se representa así:

datos[n] donde n > 0
